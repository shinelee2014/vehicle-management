## [1.0.0] - 2026-06-25

首个正式发布版本。从 0 到 1 完整搭建，覆盖核心业务 + 基础管理功能。

### 新增功能

#### 业务核心
- 进场登记（含车牌识别、车辆类型、岗亭、关联审批人）
- 出场登记（自动关联最近一条未匹配出场的进记录）
- 拍照上传（手机浏览器原生调相机）+ 后端自动加水印
- 水印内容：时间、岗亭、操作人，右下角
- 待审批列表（红点徽标自动 30 秒轮询）
- 审批流（通过/驳回，含备注，写审计日志）
- 站内信通知（提交时通知审批人，审批结果通知操作人）
- 记录查询（车牌/岗亭/类型/状态/日期多条件筛选）
- Excel 导出（≤5000 条）
- 定时任务：超时审批、1 年归档、每日日报
- 消息中心（未读红点 + 已读未读切换）
- 个人中心 + 修改密码

#### 管理后台
- 用户管理（CRUD + 改密 + 软删除）
- 岗亭管理（CRUD）
- 系统配置
- 角色权限管理（4 角色默认 + 可视化调整每个角色能看哪些模块）
- 自定义模块（admin 可新增/编辑/删除功能模块，分配给任意角色）

#### 数据治理
- 软删除记录（admin only，带原因 + 审计快照 + 可恢复）
- 完整的审计日志（每次写操作记录）

### 角色

- `security` 保安 - 进出场登记 + 自己的记录
- `supervisor` 主管 - 业务全权 + 报表
- `approver` 审批人 - 仅审批
- `admin` 管理员 - 全权

### 技术栈

- **前端**：Vue 3 + Composition API + Vite + Element Plus + Pinia + Axios
- **后端**：FastAPI + SQLAlchemy 2.0 + Pydantic + APScheduler + Pillow + python-jose + passlib
- **数据库**：MariaDB 10.11（utf8mb4）
- **部署**：Docker Compose（3 服务：db / backend / frontend）
- **目标环境**：Synology NAS DS1621+ (192.168.6.12:8080)

## 版本约定

格式：`主版本.次版本.修订号`

# 用户使用手册

> 厂区车辆进出管理系统 · 3 角色使用指南

---

## 🚪 第一部分：保安使用指南

### 登录系统
1. 打开浏览器，输入 `http://192.168.6.12:8080`（或 NAS IP）
2. 输入账号密码登录
3. 默认保安账号：`security1 / 123456`

### 进场登记
1. 左侧菜单 → **进场登记**（手机端从首页点"进场登记"）
2. 填写信息：
   - 岗亭（默认选中）
   - 车牌号（自动转大写）
   - 车辆类型（内部/外部/货车）
   - 货车需填货物描述
   - 事由/被访人
   - 选择审批人
3. 拍 2 张照片：
   - 车前
   - 车牌
4. 点击 **提交**
5. 系统自动通知审批人，等待审批

### 出场登记
1. 左侧菜单 → **出场登记**
2. 搜索车牌（输入几个字符自动补全）
3. 选择要出场的记录
4. 拍 2 张照片（车前 + 车牌）
5. 选择审批人（可与进场时不同）
6. 提交

### 查看我的记录
- 左侧菜单 → **记录查询**
- 默认只看到自己提交的记录
- 点击记录查看详情和审批进度

### 接收消息
- 右上角铃铛显示未读数量
- 点击进入消息中心
- 审批结果会通过站内消息通知

## 👔 第二部分：主管使用指南

### 登录
默认主管账号：`supervisor1 / 123456`

### 审批待办
1. 左侧菜单 → **待我审批**
2. 查看待审批列表（按时间倒序）
3. 点击 **详情** 查看完整信息（含 4 张照片）
4. 选择：
   - **✓ 通过**（可填审批意见）
   - **✗ 驳回**（必须填驳回原因）
5. 系统自动通知申请人

### 查看报表
1. 左侧菜单 → **报表中心**
2. 选择时间范围（今天/本周/本月/自定义）
3. 查看统计：
   - 总进场/总出场
   - 货车占比
   - 平均停留时长
   - 每日趋势图
4. 点击 **导出 Excel** 导出数据

### 筛选记录
- 记录查询支持多条件筛选
- 主管可看到**所有**保安的记录
- 支持按时间、车牌、岗亭、类型、状态筛选

## 👑 第三部分：管理员使用指南

### 用户管理
1. 左侧菜单 → **管理后台** → **用户管理**
2. 操作：
   - 新建用户：点 **+ 新建用户**，填账号、密码、姓名、角色、岗亭
   - 编辑用户：点击行内 **编辑** 按钮
   - 重置密码：点击 **重置密码**
   - 停用用户：编辑时取消 **启用**
3. **角色说明**：
   - 保安：只能登记、查看自己的记录
   - 主管：保安权限 + 审批 + 报表
   - 管理员：所有权限 + 用户管理

### 岗亭管理
1. 管理后台 → **岗亭管理**
2. 新建岗亭（如：3号门、仓库门）
3. 排序、停用

### 推送配置（定时报表）
1. 管理后台 → **推送配置**（下一版本）
2. 配置日报/周报
3. 设置推送时间和收件人

### 系统配置
1. 管理后台 → **系统配置**（下一版本）
2. 配置项：
   - 数据保留月数（默认 12）
   - 审批超时时间（默认 30 分钟）
   - 水印格式
   - 公司名称（报表抬头）

## 📱 第四部分：手机端使用

### 拍照技巧
- **车前照片**：站在车前 1.5-2 米，整车入镜
- **车牌照片**：距离 0.5-1 米，车牌清晰
- 系统自动加水印（时间、岗亭、操作人）

### 离线使用
- 内网部署，不依赖外网
- 没网时**不能**上传（需要拍照上传）
- 内网断网**不影响**已上传数据查询

### 添加到主屏幕（推荐）
- 浏览器 → 菜单 → **添加到主屏幕**
- 下次点击图标直接打开

## ⚠️ 第五部分：常见问题

### Q1：忘记密码怎么办？
- 联系管理员重置（管理后台 → 用户管理 → 重置密码）

### Q2：审批超时怎么办？
- 默认 30 分钟未审批 → 系统自动通过
- 可在系统配置中修改

### Q3：拍错照片怎么办？
- 暂不支持删除/替换，请重新登记一条

### Q4：审批人不在怎么办？
- 系统会自动通过（30 分钟后）
- 或联系管理员修改审批人配置

### Q5：数据保留多久？
- 在线数据：1 年
- 1 年后自动归档到 `records_archive` 表
- 管理员可手动查询归档数据

### Q6：可以导出某段时间的数据吗？
- 记录查询 → 设置时间范围 → 导出 Excel（仅主管+）

### Q7：货车登记时忘记填货物？
- 货车的"货物描述"为必填项，提交时会校验

### Q8：审批后还能改吗？
- 审批后状态锁定，不能修改
- 如需修改请联系管理员

## 🎯 最佳实践

### 保安
- 进场前提前准备好审批人（确认谁在岗）
- 照片拍清楚再上传（不要模糊的）
- 货车一定要填货物

### 主管
- 每天上班先看 **待我审批** 列表
- 及时处理审批，避免超时自动通过
- 周末/假期前调整审批人

### 管理员
- 每周查看一次报表，监控异常
- 每月备份一次数据
- 新员工入职及时建账号
- 离职员工及时停用账号
